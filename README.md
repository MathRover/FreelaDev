# FreelaDev

Simulador de orçamento para serviços de TI freelancer.

## Sobre

Ferramenta web que permite selecionar serviços de desenvolvimento e ver o valor total do orçamento em tempo real, com preços baseados no mercado brasileiro.

## Serviços disponíveis

- Landing Page
- Site Institucional
- E-commerce
- Integração de API
- Banco de Dados
- Hospedagem e Deploy
- Manutenção Mensal
- Design UI/UX

## Como usar

Abra o arquivo `index.html` no navegador.

## Tecnologias

- HTML
- CSS
- JavaScript
